# Party Mode

## Purpose

Orchestrates real-time multi-agent collaboration where 17+ specialized agents communicate simultaneously to solve complex, cross-functional problems. When a single agent perspective isn't enough, Party Mode brings the entire team into the conversation.

## Workflow: Party Mode

Dynamic orchestration of all available agents for complex problem-solving.

### Structure

1. **Initialization:** User invokes `*party-mode` with a complex problem or cross-functional query
2. **Role Selection:** System dynamically selects participating agents based on problem context
3. **Turn-Taking:** Agents contribute sequentially or conversationally based on discussion flow
4. **Synthesis:** Lead agent (Analyst or PM) synthesizes discussion into actionable outputs
5. **Resolution:** Team arrives at comprehensive solution with multiple perspectives

## Available Agents in Party Mode

**All 17+ specialized agents** participate dynamically:
- **Agile Pipeline:** analyst, product-manager, architect, developer, test-architect, scrum-master
- **Creative Intelligence:** brainstorming-coach, design-thinking-coach, innovation-oracle, master-storyteller, problem-solver
- **Security:** chief-security-officer, security-architect, security-test-analyst
- **Content & Design:** technical-writer, ux-designer
- **Specialized:** quick-flow-solo-dev

## Trigger

- Command: `*party-mode`
- Context: Complex cross-functional problems requiring multiple perspectives

## Dependencies

- Requires multiple active agents
- Relies on shared context and inter-agent communication
- Benefits from clear problem statement

## Integration Points

- **Input:** Complex problem, cross-functional query, strategic decision
- **Output:** Multi-perspective analysis, synthesized recommendations, actionable plan
- **Context:** All agent capabilities available simultaneously
- **Best For:** Strategic decisions, complex problems, innovation challenges

## Benefits

- **Multi-perspective analysis:** Get insights from 17+ specialist viewpoints
- **Faster decisions:** Parallel thinking accelerates resolution
- **Cross-functional alignment:** All teams contribute simultaneously
- **Reduced handoffs:** Direct agent-to-agent communication
- **Holistic solutions:** Account for technical, business, creative, and security angles
