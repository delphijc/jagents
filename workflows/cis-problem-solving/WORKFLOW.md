# WORKFLOW: CIS Problem Solving

**Type:** CIS Workflow
**Agents involved:** Problem Solver (Dr. Quinn)

---

## Purpose

Systematic root cause analysis to solve complex problems permanently.

## Steps

1. **Define Problem:** Clear statement of the issue.
2. **Analyze Root Cause:** Use 5 Whys or Fishbone.
3. **Generate Solutions:** Brainstorm fixes.
4. **Evaluate:** Select the best solution (Impact/Effort).
5. **Plan:** Create an action plan.

## Trigger

- Command: `*solve-problem`

## Outputs

- Cause-Effect Analysis
- Solution Action Plan
