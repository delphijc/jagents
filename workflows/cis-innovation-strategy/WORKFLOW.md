# WORKFLOW: CIS Innovation Strategy

**Type:** CIS Workflow
**Agents involved:** Innovation Oracle (Victor)

---

## Purpose

Develop strategic differentiators and disrupt existing market models.

## Steps

1. **Market Analysis:** Understand current landscape.
2. **Strategy Canvas:** Map value curves.
3. **Blue Ocean Shift:** Apply ERC grid (Eliminate, Reduce, Raise, Create).
4. **Business Model:** Define value capture.

## Trigger

- Command: `*innovate`

## Outputs

- Strategy Canvas
- Business Model Canvas
