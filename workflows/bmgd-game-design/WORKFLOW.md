# WORKFLOW: BMGD Game Design

**Type:** BMGD Workflow
**Agents involved:** Game Designer, Writer

---

## Purpose

Phase 2 of Game Development: Comprehensive Design.

## Steps

1. **GDD Creation:** Write the Game Design Document.
2. **Mechanics Design:** Define rules, systems, and interactivity.
3. **Level Design:** Plan world layout and flow.
4. **Narrative Design:** Write story, dialogue, and lore.

## Trigger

- Completion of Preproduction.

## Outputs

- Game Design Document (GDD)
- Level Plans
- Narrative Script
