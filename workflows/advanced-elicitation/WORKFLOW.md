# WORKFLOW: Advanced Elicitation

**Type:** Core Workflow
**Agents involved:** Any (Analyst, PM, Architect primarily)

---

## Purpose

Uses advanced Large Language Model (LLM) reasoning techniques to extract deeper insights, validate assumptions, and improve reasoning quality.

## Techniques Uses

1. **Chain of Thought (CoT):** Step-by-step reasoning.
2. **Self-Consistency:** Generating multiple paths and finding consensus.
3. **Devil's Advocate:** Intentionally challenging the premise.
4. **Prompt Chaining:** Breaking complex tasks into sequential prompts.

## Trigger

- Command: `*advanced-elicitation`
- Context: When an answer feels shallow or a problem requires deep reasoning.

## Outputs

- Detailed reasoning chains
- Validated conclusions
- Confidence scores
