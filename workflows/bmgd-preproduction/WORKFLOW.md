# WORKFLOW: BMGD Preproduction

**Type:** BMGD Workflow
**Agents involved:** Analyst, Game Designer (if exists) or PM

---

## Purpose

Phase 1 of Game Development: Conceptualization and Prototyping.

## Steps

1. **Brainstorm Game:** Using game-specific ideation techniques.
2. **Game Brief:** Define core loop, genre, platform, and target audience.
3. **Market Analysis:** Compare with existing games.

## Trigger

- Context: New game project.

## Outputs

- Game Brief
- Concept Art (Description)
- Prototype Plan
