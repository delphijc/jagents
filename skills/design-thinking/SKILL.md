# SKILL: Design Thinking

**Purpose:** Human-centered design and user experience optimization  
**Domain:** UX design, empathy mapping, user flows  
**Type:** Creative Intelligence Suite (CIS) Module

---

## Overview

The Design Thinking skill applies human-centered design principles to ensure products solve real user problems effectively. It focuses on empathy, ideation, and user-centric approaches.

---

## Core Capabilities

1. **Persona Mapping**
   - User persona development
   - Empathy mapping
   - User journey analysis

2. **Empathy Checks**
   - User pain point identification
   - Needs assessment
   - Constraint analysis

3. **User Flow Definition**
   - Journey mapping
   - Interaction design
   - Navigation optimization

4. **Visual Standards**
   - UI/UX philosophy establishment
   - Style guide creation
   - Accessibility planning

---

## Design Thinking Process

### 1. Empathize
- Understand user needs
- Map user personas
- Identify pain points

### 2. Define
- Problem statement creation
- User needs prioritization
- Constraint identification

### 3. Ideate
- Solution brainstorming
- Alternative generation
- Creative exploration

### 4. Prototype
- Wireframe creation
- User flow modeling
- Interface mockups

### 5. Test
- User feedback collection
- Iteration planning
- Validation
