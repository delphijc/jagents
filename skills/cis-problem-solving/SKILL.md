# SKILL: Problem Solving

**Purpose:** Systematic root cause analysis and solution generation
**Domain:** Analytical Thinking
**Type:** Creative Intelligence Suite (CIS) Module

---

## Overview

The Problem Solving skill extracts the root causes of complex issues using structured analytical frameworks, moving beyond symptom treatment to permanent resolution.

---

## Core Capabilities

1. **Root Cause Analysis**

   - **5 Whys:** Iterative interrogative technique
   - **Fishbone Diagram (Ishikawa):** Cause-and-effect visualization

2. **Solution Evaluation**

   - Impact vs. Effort Matrix
   - SWOT Analysis

3. **Decision Making**
   - Pugh Matrix
   - Decision Trees

---

## Integration Points

- **Input:** Problem statement, symptoms
- **Output:** Root cause identification, solution recommendations
- **Used By:** [Problem Solver](../../agents/cis-problem-solving-coach/AGENT.md), [Architect](../../agents/architect/AGENT.md), [Developer](../../agents/developer/AGENT.md)
