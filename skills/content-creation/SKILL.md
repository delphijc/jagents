# SKILL: Content Creation & Blogging

**Purpose:** Structured content creation and blog post generation  
**Domain:** Writing, essay formatting, content marketing  
**Type:** Reusable Module (Command/Fabric Pattern)

---

## Overview

The Content Creation & Blogging skill provides structured approaches to content creation using Markdown/Text orchestration. Following the **UNIX philosophy**, this skill solves content problems once and turns them into reusable modules.

---

## Core Capabilities

1. **Blog Post Creation**
   - Essay dictation to formatted blog post
   - Style guide application
   - Consistent formatting

2. **Essay Writing**
   - Structured essay composition
   - Argument development
   - Citation management

3. **Content Formatting**
   - Markdown formatting
   - Style guide compliance
   - Brand voice consistency

4. **Content Optimization**
   - SEO optimization
   - Readability enhancement
   - Audience targeting

---

## Commands & Fabric Patterns

### Primary Commands

- `write-blog` - Format dictated essays into blog posts using project style guidelines
- `write-essay` - Create structured essays
- `format-content` - Apply formatting standards

### Fabric Patterns

Utilizes reusable Fabric patterns for:
- Content structure
- Writing style
- Formatting rules
- Brand voice

---

## Execution Pattern

1. **Input Collection**
   - Receive raw content or dictated text
   - Load project style guidelines from context

2. **Content Processing**
   - Apply structural templates
   - Enforce style guide rules
   - Optimize for readability

3. **Formatting Application**
   - Markdown formatting
   - Header hierarchy
   - Visual elements

4. **Output Generation**
   - Publication-ready content
   - Style-compliant formatting
   - Brand-aligned voice

---

## Context Requirements

- Project style guidelines
- Brand voice parameters
- Target audience specifications
- SEO requirements (if applicable)

---

## Tool Chaining

This skill can be chained with:
- **Image Creation Skill** - Generate header images

## Integration Points

- **Input:** Raw text, dictated content, topic outline
- **Output:** Formatted blog posts, essays, articles
- **Context:** Style guidelines, brand voice, audience parameters
- **Tools:** Markdown processors, style checkers, SEO analyzers
